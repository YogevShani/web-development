Understanding the Grid View (Float Based 12 Column Layout)

1. What is Grid View

Grid View divides a web page into a series of columns. Each element can take up a number of these columns. A common layout uses 12 columns because it can be divided into many useful combinations such as 3, 4, 6 and 12.

This page uses a classic float based 12 column system. Each column width is defined as a percentage so the layout becomes flexible and grows or shrinks based on screen size.

2. The 12 Column System Used Here


*{box-sizing:border-box;}
[class*="col-"] {
    float: left;
    border: 1px solid black;
}

.col-1 {width: 8.33%;}
.col-2 {width: 16.66%;}
.col-3 {width: 25%;}
.col-4 {width: 33.33%;}
.col-5 {width: 41.66%;}
.col-6 {width: 50%;}
.col-7 {width: 58.33%;}
.col-8 {width: 66.66%;}
.col-9 {width: 75%;}
.col-10 {width: 83.33%;}
.col-11 {width: 91.66%;}
.col-12 {width: 100%;}

Each class sets a width based on a percentage of the full row. The float left rule makes the columns sit next to each other in the same row. When the row reaches 12 columns in total width, the next element moves to a new row.

6. Summary

  • The layout is divided into 12 equal columns
  • Each element uses col classes to span a number of columns
  • Widths are percentages so the layout fits any screen size
  • float left places the columns next to each other
  • Rows reset when elements reach 12 columns in total
