CSS Flexbox Layout Module

Topic: What is Flexbox?

Flexbox is a modern CSS layout system that gives us full control over the direction, alignment, spacing, and distribution of elements, without using float or tricky positioning.

Before activating Flexbox, we start with a simple block layout like the example below.

Base HTML Structure

<div class="container">
    <div class="box">This is block1</div>
    <div class="box">This is block2</div>
    <div class="box">This is block3</div>
</div>

This is our starting point. All three blocks appear one under the other because they are normal block elements.

Base CSS

.container{
    width:60%;
    margin: auto;
    background-color:red;
}

.box{
    box-sizing:border-box;
    border-style:solid;
    font-size:30px;
}

The container is centered and each box gets a border and large text. This will help us clearly see changes once Flexbox is added.

Right now, everything stacks vertically, which is the default behavior before Flexbox is activated.

Step 1: Activating Flexbox with display:flex

To turn the container into a Flexbox container, we simply add display:flex to the existing .container class. The moment we do this, the three blocks stop stacking vertically and move into a single row.

Updated HTML

<div class="container">
    <div class="box">This is block1</div>
    <div class="box">This is block2</div>
    <div class="box">This is block3</div>
</div>

Updated CSS (only the change)

.container{
    width:60%;
    margin:auto;
    background-color:red;
    display:flex;        /* Flexbox is now active */
}

This single property transforms the layout: all boxes now align next to each other in a horizontal row since flex-direction defaults to row.

As soon as Flexbox is enabled, the items flow horizontally instead of stacking vertically. This is the core behavior of Flexbox.

Step 2: Changing Direction with flex-direction: column

By default, Flexbox arranges items in a horizontal row. To stack the items vertically using Flexbox, we change the direction to column. This gives us full control over vertical layouts without removing Flexbox behavior.

Updated CSS (only the change)

.container{
    width:60%;
    margin:auto;
    background-color:red;
    display:flex;
    flex-direction:column;    /* Items stack vertically */
}

Now the Flexbox container behaves like a vertical layout engine, stacking all three items from top to bottom.

HTML Structure (unchanged)

<div class="container">
    <div class="box">This is block1</div>
    <div class="box">This is block2</div>
    <div class="box">This is block3</div>
</div>

Flex Direction: row-reverse

The property flex-direction: row-reverse flips the order of the flex items horizontally. The layout direction becomes reversed, meaning the last item appears first on the right side.

Container CSS

.container{
    width:60%;
    margin:auto;
    background-color:red;

    display:flex;
    flex-direction: row-reverse;
}

Box CSS

.box{
    border:1px solid black;
    font-size:20px;
    padding:10px;
    box-sizing:border-box;
}

Step 3: Allowing Items to Move to New Lines with flex-wrap: wrap

By default, Flexbox keeps all items on one line, even if they overflow the container. When we want items to move to the next row (responsive layout), we use flex-wrap: wrap. This tells Flexbox that items are allowed to break into additional lines.

Updated CSS (only the change)

.container{
    width:60%;
    margin:auto;
    background-color:red;
    display:flex;
    flex-wrap:wrap;      /* Items can move to new rows */
}

As soon as wrapping is enabled, items will automatically jump to the next line when they run out of horizontal space. This is one of the most important tools for building responsive layouts.

HTML Structure (unchanged)

<div class="container">
    <div class="box">This is block1</div>
    <div class="box">This is block2</div>
    <div class="box">This is block3</div>
</div>

Step 4: Horizontal Alignment with justify-content: center

The property justify-content controls how items are positioned horizontally inside a Flexbox container. Setting it to center moves all items to the middle of the row.

Updated CSS (only the change)

.container{
    width:60%;
    margin:auto;
    background-color:red;
    display:flex;
    justify-content:center;    /* Items are centered horizontally */
}

All items now sit together in the center of the line.

Step 5: Spacing with justify-content: space-between

space-between spreads items evenly across the row. The first item sticks to the left, the last item sticks to the right, and all remaining space is divided equally between the items.

Updated CSS (only the change)

.container{
    width:60%;
    margin:auto;
    background-color:red;
    display:flex;
    justify-content:space-between;  /* Items spread to edges */
}

This is commonly used in navigation bars and layouts that need equal distribution.

Step 6: Spacing with justify-content: space-around

space-around gives equal space around each item. This means the edges of the container also receive half-sized spacing, creating a softer and more balanced layout.

Updated CSS (only the change)

.container{
    width:60%;
    margin:auto;
    background-color:red;
    display:flex;
    justify-content:space-around;   /* Equal space around items */
}

This creates even spacing that feels more breathable compared to space-between.

Align Items (Cross-Axis Alignment)

The align-items property controls how Flexbox items are aligned along the cross axis. In a horizontal flex container (the default), the cross axis is vertical. Each value changes how items are positioned inside the container's height.

Below are all available values of align-items, each illustrated visually. 

align-items: flex-start;   /* Items at the top */
align-items: flex-end;     /* Items at the bottom */
align-items: center;       /* Items centered vertically */
align-items: stretch;      /* Items stretch to fill container (default) */
align-items: baseline;     /* Items align according to text baseline */

align-items: flex-start

Items align at the top of the container (start of the cross-axis).

.container{
    display:flex;
    align-items:flex-start;
    height:200px;
}

Flex Grow (flex: 1)

The property flex: 1 tells a flex item to grow and take all remaining free space inside the container. If multiple items have flex:1, they share the space equally.

Container CSS

.container{
    width:60%;
    margin:auto;
    background-color:red;

    display:flex;
}

Box CSS

.box{
    border:1px solid black;
    box-sizing:border-box;
    padding:10px;
}

.box-grow{
    flex:1;   /* grows to fill leftover space */
}

Gap Between Flex Items (gap)

The property gap creates spacing between flex items only. It does not add space before the first item or after the last item. It is a clean alternative to using margins on each item.

Container CSS

.container{
    width:60%;
    margin:auto;
    background-color:red;

    display:flex;
    gap: 20px;   /* space between items */
}

Box CSS

.box{
    border:1px solid black;
    padding:10px;
    box-sizing:border-box;
}
