Lesson 2, CSS Cheat Sheet

Category Property / Selector Usage Example
Selectors p Selects all <p> tags p { color: blue; }
Selectors .intro Selects elements with class="intro" .intro { font-size: 20px; }
Selectors #main Selects element with id="main" #main { text-align: center; }
Selectors h1, p Groups multiple selectors h1, p { color: green; }
Selectors input[type="text"] Attribute selector input[type="text"] { border: 1px solid black; }
Text text-align Aligns text p { text-align: center; }
Text text-decoration Underline, remove underline, etc a { text-decoration: none; }
Text text-transform Uppercase, lowercase, capitalize h1 { text-transform: uppercase; }
Text text-indent Indent first line p { text-indent: 20px; }
Text letter-spacing Space between letters p { letter-spacing: 2px; }
Text word-spacing Space between words p { word-spacing: 5px; }
Text line-height Space between lines p { line-height: 1.5; }
Fonts font-family Font type p { font-family: Arial, "Times New Roman", serif; }
Fonts font-style Normal, italic, oblique p { font-style: italic; }
Fonts font-weight Thickness of text p { font-weight: bold; }
Fonts font-size Text size p { font-size: 16px; }
Fonts color Text color p { color: red; }
Colors Named colors Use names like red, blue h1 { color: green; }
Colors Hex colors Use hex codes h1 { color: #FF0000; }
Colors RGB / RGBA Use rgb values h1 { color: rgb(255,0,0); }
Backgrounds background-color Set background color body { background-color: yellow; }
Backgrounds background-image Add background image div { background-image: url("pic.jpg"); }
Backgrounds background-repeat Repeat pattern or not div { background-repeat: no-repeat; }
Backgrounds background-position Position image div { background-position: center; }
Backgrounds background-size Size of image div { background-size: contain; }
Units px Absolute pixels p { font-size: 20px; }
Units em Relative to parent font size p { font-size: 2em; }
Units % Relative to parent element p { width: 80%; }
Cascade / Precedence Inline > id > class > tag Defines which CSS rule wins Inline style overrides all
