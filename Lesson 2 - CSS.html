What Can CSS Change?

  • Text color, size, font, and alignment
  • Backgrounds and images
  • Borders, margins, and spacing
  • Tables, links, and lists
  • Layout (display, position, float, etc.)

2. Text Properties


.text-example {
  text-align: center;
  text-decoration: underline;
  text-transform: uppercase;
  text-indent: 20px;
  letter-spacing: 2px;
  word-spacing: 5px;
  line-height: 1.8;
}
CSS controls alignment, decoration, capitalization, spacing, and line height.

6. Units of Measurement


.unit-px { font-size: 20px; }
.unit-em { font-size: 2em; }
.unit-percent { width: 50%; background-color: lightblue; }
Font size = 20px (absolute)
Font size = 2em (relative to parent)
Width = 50% (relative to parent container)


7. Cascade and Precedence

This text is blue because inline style wins over class and ID.
