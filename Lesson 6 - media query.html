Media Queries, Responsive Design in Action

1. What Are Media Queries

Media Queries allow us to change the design of the page according to the screen width. This is what makes websites responsive.

Instead of one fixed design for all devices, the layout adapts automatically.

2. Basic Media Query Structure


@media screen and (max-width: 1000px) {
    /* Styles for tablets */
}

@media screen and (max-width: 800px) {
    /* Styles for mobile phones */
}

max width 1000 pixels activates the rules for tablets
max width 800 pixels activates the rules for mobile screens

3. Media Query Example


@media screen and (max-width: 1000px) {
    li {
        width: 50%;
    }
}

@media screen and (max-width: 800px) {
    li {
        width: 100%;
    }

    span {
        display: none;
    }

    .cakes {
        height: 100px;
    }
}

5. What Happens at Each Screen Size

  • Above 1000px, the menu shows 4 items in one row
  • Below 1000px, the menu changes to 2 items per row
  • Below 800px, the menu changes to 1 item per row
  • The span text disappears
  • The cakes block becomes shorter

6. Summary, Media Queries

  • Media Queries change layout based on screen size
  • They control width, height, and visibility
  • They are the core tool for responsive web design
  • They work automatically, no JavaScript needed
