CSS Position Test

Below is the CSS code applied to the red box:

.box-wrong {
  width: 150px;
  height: 100px;
  background-color: lightcoral;
  left: 150px;   }

Question: Why is the red box not moving even though we used left: 150px

The Answer

The property left works only when the element has a position value such as relative, absolute or fixed. Because we did not set any position, the browser uses the default value which is position: static, and in static mode the properties top, left, right and bottom have no effect. This is why the red box does not move.

Correct CSS:

.box-correct {
  width: 150px;
  height: 100px;
  background-color: lightgreen;

  position: relative;
  left: 150px;
}
