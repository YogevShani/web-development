Building a Horizontal Navbar Step by Step

1. Basic Navbar Without Design

First, we build a navbar using only HTML. Right now, there is **no CSS at all**, so the browser shows a simple vertical list.

This is how the navbar looks without design. 

<nav>
    <ul>
        <li><a href="category1.html">Home Page</a></li>
        <li><a href="contact.html">Contact</a></li>
        <li><a href="about.html">About US</a></li>
        <li><a href="products.html">Products</a></li>
    </ul>
</nav>
  • Home Page
  • Contact
  • About US
  • Products

The browser automatically shows this as a vertical menu with bullet points.

2. The Same Navbar With CSS Applied

Now we apply CSS to design the menu so links will look like buttons.

3. Removing the Bullet Points


ul {
    list-style-type: none;
}

By default, the browser places dots before each list item. This rule removes them so the menu looks clean.

  • Contact

4. Making the Menu Horizontal


li {
    width: 10%;
    float: left;
}

float left moves all menu buttons into one horizontal row. width 10 percent controls how much horizontal space each button gets from its parent container's width.

5. Styling the Links as Buttons


a {
    display: block;
    text-decoration: none;
    color: white;
    padding: 10px;
    text-align: center;
    background: url("menu-background-3.jpg") no-repeat;
}

text decoration none removes the underline color white changes the text color padding makes the button larger background adds an image to the button

6. Improve the links look like Buttons


a {
    display: block;
    border-radius: 10px;
    margin: 2px;
}

display block makes the entire button clickable border-radius to round the button corners margin to make small space between buttons

7. Adding Hover Effect


a:hover {
    background: url("menu-background-4.jpg") no-repeat;
}

When the mouse is placed over a button, the background image changes. This makes the menu interactive and user friendly.

8. Summary

  • The navbar is built using a list of links
  • ul removes dots using list style type none
  • li becomes horizontal using float left
  • a is styled like a button
  • a hover adds mouse interaction
