Lesson 1, HTML Tags Cheat Sheet

Tag Usage Example
<ul> Unordered, bulleted list <ul><li>Apple</li><li>Banana</li></ul>
<ol> Ordered, numbered list <ol><li>First</li><li>Second</li></ol>
<li> List item inside <ul> or <ol> <li>Item</li>
<dl> Description list <dl><dt>HTML</dt><dd>Markup language</dd></dl>
<dt> Term in description list <dt>CSS</dt>
<dd> Definition in description list <dd>Styles web pages</dd>
<img> Display an image <img src="panda.jpg" alt="Panda" width="200">
<table> Define a table <table>...</table>
<tr> Table row <tr><td>Row1</td></tr>
<td> Table cell <td>Value</td>
<th> Header cell <th>Name</th>
<thead> Group of header rows <thead><tr><th>Title</th></tr></thead>
<tbody> Group of body rows <tbody><tr><td>Data</td></tr></tbody>
<tfoot> Group of footer rows <tfoot><tr><td>Summary</td></tr></tfoot>
<a> Hyperlink <a href="https://google.com">Google</a>
href Link target attribute <a href="about.html">About</a>
target="_blank" Open link in new tab <a href="..." target="_blank">Link</a>
id Internal anchor target <h2 id="tips">Tips</h2>
#id in href Jump to element on same page <a href="#tips">Go to Tips</a>
<iframe> Embed webpage <iframe src="https://example.com" width="600" height="400"></iframe>
Entities Special characters &nbsp;, &lt;, &gt;, &amp;, &quot;, &apos;, &copy;, &reg;
