Images

<img> displays an image. It uses src for the file location and alt for alternate text.

Image Formats

  • JPG / JPEG – Best for photos, supports millions of colors, small file size.
  • GIF – Supports animation and transparency but only 256 colors.
  • PNG – Supports high-quality images and full transparency, larger file size.
