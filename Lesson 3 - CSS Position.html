CSS Position Demonstration

This example shows static, relative, absolute, fixed, sticky and how overlays work with absolute positioning.

1️⃣ position: static (default)

Static

Static is the default. The element remains in normal document flow and ignores top/left/right.

CSS used:

.static { background-color: #6aa84f; position: static; }

2️⃣ position: relative

Relative

Relative keeps the element in the normal flow, but shifts it via top/left.

CSS used:

.relative { background-color: #3d85c6; position: relative; top: 20px; left: 40px; }

3️⃣ position: absolute

Container (relative)
Absolute

Absolute removes the element from flow and positions it relative to the first ancestor with position not static.

CSS used:

.absolute-container { position: relative; background-color: #ddd; height: 150px; border: 2px dashed gray; } .absolute { background-color: #e69138; position: absolute; top: 30px; right: 20px; }

Overlay Examples (Absolute Overlay + Background Image)

These examples demonstrate how to place text on top of an image using either absolute positioning or a background image.

Example A – Image + position:absolute overlay

  • Image is real content (<img> element).
  • Overlay is placed above it using absolute positioning.
  • Good for content images.
  • Requires a parent with position: relative.

Example B – background-image overlay

  • Image is decorative, provided through CSS.

4️⃣ position: fixed

Fixed Box

Fixed elements stay attached to the viewport.

CSS used:

.fixed { background-color: #cc0000; position: fixed; bottom: 10px; right: 10px; width: 200px; height: 60px; z-index: 10; }

5️⃣ position: sticky

Sticky behaves like relative until reaching its threshold then sticks.

CSS used:

.sticky { background-color: #674ea7; position: sticky; top: 0; }
