CSS Display Test

Below is the CSS code applied to the red box:

.box-wrong {
  display: inline;
  width: 200px;   
  height: 80px;
  background-color: lightcoral;
}

Question: Why is the width not changing even though we set width: 200px

Red Box


The Answer

The width property does not work on inline elements.
The div was set to display: inline so the browser ignores the width value and the element shrinks to fit its content.
A div normally has a default display value of block, and in block mode width works.
To allow the width property to work here, the element can be inline-block or block.

Correct CSS:

.box-correct {
  display: inline-block;
  width: 200px;
  height: 80px;
  background-color: lightgreen;
}
